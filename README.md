# Mall-Customer-Data-Cleaning
Data Cleaning and Visualization Project using the Mall Customers Dataset. This project demonstrates data preprocessing, duplicate and missing value checks, outlier detection, and data visualization using Python, Pandas, Matplotlib, and Seaborn.
# 🛍️ Mall Customer Data Cleaning & Visualization

## 📌 Project Overview

This project focuses on cleaning, analyzing, and visualizing the Mall Customers dataset using Python. The objective is to understand the dataset, perform data preprocessing, and create meaningful visualizations to gain business insights.

---

## 🎯 Objective

- Load and explore the dataset.
- Check and handle missing values.
- Identify and remove duplicate records.
- Detect outliers using boxplots.
- Visualize customer information using different charts.
- Save the cleaned dataset.

---

## 🛠️ Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

---

## 📂 Dataset

**Dataset Name:** Mall Customers Dataset

### Features

- CustomerID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

---

## 📊 Data Cleaning Steps

- Imported the dataset using Pandas.
- Checked the dataset structure.
- Verified missing values.
- Checked duplicate records.
- Identified outliers using boxplots.
- Saved the cleaned dataset.

---

## 📈 Data Visualization

The following visualizations were created:

- Gender Distribution
- Age Distribution
- Annual Income Distribution
- Spending Score Distribution
- Scatter Plot (Annual Income vs Spending Score)
- Correlation Heatmap

---

## 🔍 Key Findings

- No missing values were found in the dataset.
- No duplicate records were identified.
- Customer ages vary across different age groups.
- Spending scores differ among customers.
- Annual income and spending score do not show a strong linear relationship.
